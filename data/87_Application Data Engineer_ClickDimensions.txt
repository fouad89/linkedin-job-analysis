Application Data Engineer
 ClickDimensions is looking for an Application Data Engineer to join the team on a full-time basis. We are looking for a technical individual to own and manage the execution of the daily data pipelines operations. In undertaking this challenge, you will use the latest distributed computing technologies to run our data pipeline processes on large-scale data across multiple Public Clouds. These pipeline processes span ingesting, staging, cleansing, deduplicating, mapping, and publishing. You will also oversee the provisioning new customers, loading and cleansing the historical data.
  Key Objectives/Duties:
·        Coordinating customer provisioning 
·        Communicating with different teams to orchestrate the process whenever is needed
·        Building automated reports 
·        Shaping dashboards that bring clarity around operational status, health and COGS. 
·        Daily health and issue triage 
·        Triage and resolve data pipeline issues 
·        Run data quality reports and perform data improvement interventions 
·        Monitoring and alert detection 
·        Monitor the progress of data flow and report status to a broader team 
·        Desing and help preemptive alert system based on data sampling 

Requirements:
·        Solid programming and engineering background. 
·        +2 years' experience programming with Python and SQL. 
·        +2 year experience using cloud services, preferably Azure. 
·        +2 year experience using monitoring tools, either Grafana, LogAnalytics, Kibana or similar. 
·        +2 year experience using PowerBI. 
·        Strong verbal and written communication skills. 
·        Strong analytical, organizational, and problem-solving skills. 
·        Self-motivated and results-oriented person. 
·        Worked remotely in the past. 

Nice to have: 
·        Acted as a project manager or team lead, showing skills to prioritize and organize tasks across different people. 
·        Some Azure Databricks, Datafactory experience or exposure. 
·        Experience modeling data. 
·        Experience building monitoring artifacts, including the usage of Kusto.
